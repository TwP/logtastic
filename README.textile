h2. Logging CouchDB

* by Tim Pease
* http://github.com/TwP/logging-couchdb/tree/master

h3. Description

Brings together CouchDB with the Log4(X) style logging frameworks. The goal is
to collect log messages from most any application and display those messages in
a convenient web interface. Some nice plots and summary data is provided.

h3. Features

Provides a couchapp for viewing log messages from various applications.
Provides logging appenders for the following programming languages:

* Ruby via the Logging framework
* Java via the Log4r framework (in progress)

Contributions for other languages are welcome.
